## Списки

Списки ненуменрованные выделяются одной звездочкой. Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Так же есть списки нумерованные. Через нумерацию. Вот так
1. Элемент 1
2. Элемент 2
3. Элемент 3
